## 组件打包工具 支持 uniapp

### 下载

```markdown
npm i keylion-cli -g
```

### 使用

#### Compile component library

```shell
# playground-ignore
keylion-cli compile
```

#### Build a documentation site

```shell
# playground-ignore
keylion-cli build
```

#### Start the development server

```shell
# playground-ignore
keylion-cli dev
# force mode
keylion-cli dev -f

###

```

#### Generate a project template

```shell
# playground-ignore
keylion-cli gen
```

#### Create a component template file

```shell
# playground-ignore
keylion-cli create
```
